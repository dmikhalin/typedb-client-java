[#_UserManager]

Provides access to all user management methods.

[#_all]
==== all

[source,nodejs]
----
all(): Promise<User[]>
----

Retrieves all users which exist on the TypeDB server.

.Returns
`Promise<User[]>`

.Code examples
[source,nodejs]
----
driver.users.all()
----
