[#_Transitivity]
= Transitivity

== Description

This class is used for specifying whether we need explicit or transitive subtyping, instances, etc.

== Code examples

[source,python]
----
attribute_type.get_owners(transaction, annotations=annotations, transitivity=Transitivity.EXPLICIT)
----
